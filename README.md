# AstonCV

## Overview
A web application built using PHP and MySQL that allows users to create, view and search CV profiles. This project was developed as part of my university's "Internet Applications and Databases" module.

## Features
- User registration and login system
- Password reset functionality
- Create and update CVs
- Search CVs by first name, last name or programming language
- Guest and logged-in user views
- View CV details

## Technologies Used
- PHP
- MySQL
- PDO
- HTML
- CSS

## Security Features
- Password hashing
- SQL injection prevention using PDO
- XSS prevention
- Form validation
- Authentication and authorisation

## System Structure
The application follows a three-tier architecture:

- **Presentation Layer:** HTML and CSS for the user interface
- **Application Layer:** PHP for authentication, validation and database interactions
- **Data Layer:** MySQL database
